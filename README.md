School Management System (Python)

A simple Python-based School Management System that manages students, courses, grades, attendance, and generates reports exported to Excel.

Features

Add students and courses
Enroll students in courses
Record grades and attendance
Calculate total score, average, attendance percentage
PASS / FAIL status per subject
Export student reports to Excel

Concepts Used

Object-Oriented Programming (OOP)
Dictionaries & Lists
User input handling
Excel export using openpyxl

Requirements

Python 3.7+
openpyxl

Run the Project

python main.py

Output

Console student reports
student_reports.xlsx file with all student data
